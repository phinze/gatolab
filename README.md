# gatolab

Home base for hacking on my Elgato Stream Deck Plus.

## Why "gatolab"?

"El Gato" means "The Cat" in Spanish - a nod to Elgato while avoiding the Steam Deck / Stream Deck naming collision.

## Hardware

- Elgato Stream Deck Plus (8 LCD keys + 4 rotary dials + touch strip)
